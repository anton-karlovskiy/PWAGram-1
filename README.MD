Instagram clonish built using the principles of progressive web app

-Device camera

-Location API

-Reverse Geolocation (LocationIQ)

-Push Notifications (Push Manager)

-Service workers

-Offline Mode (Also for syncing new posts)

-Caching

-Indexed DB
